# Figma Designer System

## Company Identity

- **Industry:** Investment, Urban Landscape & Exterior Design
- **Background:** Interior Architecture & Exhibition Design
- **Orientation:** Progressive, growth-driven, innovation-focused
- **Design Preference:** Soft, curved, fluid forms over rigid straight lines

## Design Directives

### Page Structure

- First page = Hero Section with strong visual impact
- Hero must immediately communicate brand identity and vision

### Visual Priority

- Prioritize icons, illustrations, and imagery over long text
- When text is used: Short, Highly emphasized, Message-driven

### Form Language

- Curved lines and organic shapes as primary structural elements
- Avoid excessive straight or sharp geometry

### Spatial Expression

- Prefer three-dimensional and layered designs over flat layouts
- Create depth, scale, and spatial expansion

### Motion & Interaction

- Avoid static composition
- Use animation and transitions to express activity, innovation, and flow

### Layout & Spacing

- Apply wide spacing between components
- Overall feel: Open, Breathable, Premium, Unconstrained

## Quick Reference

| Principle | Rule                                    |
| --------- | --------------------------------------- |
| Form      | Curved, organic, fluid — no sharp edges |
| Space     | Wide margins, breathing room, premium   |
| Depth     | Layered, 3D, dimensional                |
| Motion    | Animated, dynamic, never static         |
| Content   | Visual-first, minimal text              |
| Brand     | Progressive, innovative, spatial        |
